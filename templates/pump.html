WATER PUMP SYSTEM

Description

The system in focus is the Water Pump System which pumps water in a specified area of the city. This system predicts Water Pump module failure to allow faster repairs before the entire area's water supply comes to a halt.

Input Description: The input to the system must contain 51 comma-separated feature values recorded from the Water Pump sensors.

API

Water pump status can be checked remotely using the API. To use the API, use the following HTTP GET Request URI:

http://<server-ip>:<server-port>/predict?model=pump&data=<sensor-data>

The API can return 3 values: "0" : FAILURE || "1" : NORMAL || "None" : INVALID INPUT
